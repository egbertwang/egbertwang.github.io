2019计划

发表于 2019-02-01 更新于 2019-02-04 分类于 生活

填坑

2018 的计划几乎都没完成，讲道理 2018 什么都没做，花了很多功夫在没用的地方。2018 的计划主要集中在 OpenGL、Unity 3D、Vue。前半年计划做的游戏，由于懒惰并没有完成。还有 Vue，是出于对工作的期望。本来想可以学一下 Vue，来保证自己未来找得到工作，却因为安装环境出了问题，一直没成功。其实现在一直都是瞎猫摸死耗子，没做成事还是心不诚，没有真正的想做事情。

计划

今年有两个阶段，第一个阶段是上半年，主要是要完成学业，做好毕业设计，并且找到工作。毕业设计已经开始，并且按照计划进行。学业的话要付出一些努力，这次就是背水一战了。找工作的事情也得准备。假期的时候就要赶紧学Java。必须掌握足够的技能。

另外是自己的计划。我自己有很多项目要做。首先是图像拼接的软件，这个要在3月份内做完。还有一个游戏，我准备在5月份做完。这只是一个简单的游戏。大游戏的话可能没什么机会了。

还有心愿单，我想买的东西。

  • Nas 存储
  • 树莓派
  • 耳机
  • 键盘

无非是这些东西，玩而已。
