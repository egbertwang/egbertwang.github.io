大胆的计划

发表于 2017-12-23 分类于 日常

好久了，是时候总结一下了，顺便展望一下。

回顾

之前说好的新的开始其实也 并没有什么改变，依然很废。考试还是很慌。但是还是学到了一些东西的。

做过的事情

Android 开发

由于软工实践接触到了我最讨厌的 Android。可能是因为国内 Android 环境太差了，所以一直很抵触 Android 手机和 Android 开发。但是这次还是接触到了。花了很长时间来了解相关知识，现在也基本上手了。掌握的一些方法或者说是工具包括 okhttp, Gson 等。自己贡献的代码还是不够。最后一个星期希望可以写出好的代码。

WebGL 开发

因为自己的游戏梦所以报了图形学这门选修。不知道是坑还是福，总之不知道自己在学什么，作业也很困难。之前买的 OpenGL 的书也没怎么看，大概是要凉了吧。希望自己可以捡起来这一块。

MATLAB 上的 BP 神经网络编程

这是一次机缘巧合，帮助好友完成一个论文，做一点微小的工作。正好自己在复习人工智能，就正好了解一下，还是有有点困难的，准备放在周末做。希望周末可以做完吧，毕竟我答应的事哪有完成不了的。

未完成的事情

作业

Sudoku 这个作业之前就说好的要做完，可是臣妾真的做不到啊，只能尽力了。华为软件云的真的要在周末完成啊。

展望

服务器

之前租了一年阿里云的服务器，由于一直不会做，而且没时间，就一直耽搁了。寒假要学一下 PHP 和 python 了，尽快把服务器做好，把自己的网站建起来，自己的 App 也要用到这个服务器的。所以一定要尽快。

Self Management App

一个功能全面的自我管理软件，跨平台云同步。功能基本上决定了，就剩一些细节要考虑，还在策划中…

AI 机器人

这个真的是终极计划了。会用到 3D 打印之类的东西，还在想它的定位以及我心目中的功能。可以说是非常厉害了。这个是我真的要做的东西，现在要开始攒钱了。
